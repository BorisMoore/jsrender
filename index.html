Basic Usage

Compile template object from string

You can create templates by using the $.templates() command.

In the moment you want to render a template you simply have to call render() on a specific template.

Compile from string as named template

If you want to store a template inside of JsRender simply call $.templates() providing the name of your template and the template itself as arguments to register it.

Once your template is registered it is possible to render it by just calling $.render.nameOfYourTemplate() or $.render["nameOfYourTemplate"]().

Render template from script block

You can store your templates in script blocks inside your html document identified by the 'id' attribute of that block.

After selecting a specific template with jQuery you can call the render(data) method over that to recive your rendered template as string.

Basic Commands

Outputting data
containing HTML markup

For default data output use the {{:value}} command.

Outputting data
HTML-escaped

For outputting data that should be HTML-escaped you can use the {{>value}} command.

Itterating Arrays

Itterating an array

With the {{for myArray}} command you can itterate over the entries of an array. Close the itteration-block with an ending {{/for}}.
