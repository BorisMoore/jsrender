JsRender Demos

To pass an array to a "layout template" - which includes headers or footers,
as well as the rendered items - wrap the array in an array...

Top-level layout:
$( "#movieList" ).html(
    // Wrap movies array in an array
    $("#movieTemplate").render( [movies] )
);

Template:

   header
   {{for #data}}
      item
   {{/for}}
   footer
Nested layout:
{{for [languages]}}
    header
    {{for #data}}
        item
    {{/for}}
    footer
{{/for}}
