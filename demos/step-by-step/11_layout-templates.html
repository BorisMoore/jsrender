JsRender Demos

Using layout templates to render arrays along with headers or footers.

Top-level layout:
$.templates( "moviesLayout", {
	markup: "#movieTemplate",
	layout: true
});

$( "#movieList" ).html(
	$.render.moviesLayout( movies )
);

header
	{{for #data}}
		item
	{{/for}}
footer
Nested layout:
{{for languages layout=true}}
	header
	{{for #data}}
		item
	{{/for}}
	footer
{{/for}}
